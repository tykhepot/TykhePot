# TykhePot 白皮书

## 摘要

TykhePot ($TPOT) 是基于 Solana 区块链的公平透明娱乐协议。通过双奖池设计、通缩机制和普惠参与模式，创造一个可持续的链上娱乐生态系统。

## 1. 项目背景

### 1.1 命名由来
Tykhe（堤喀）是古希腊神话中的幸运女神，掌管命运与机遇。正如她的名字所寓意，TykhePot 致力于为每个参与者提供公平的幸运机会。

### 1.2 市场痛点
- 传统彩票缺乏透明度
- 高额手续费和运营成本
- 缺乏可持续的经济模型

```
收益 = 本金 × 年化率 × 天数 / 365
```

## 7. 空投机制

### 7.1 核心设计
- **空投免费领** - 每人固定额度
- **代币锁定** - 空投代币只能参与游戏，不能直接转账/交易
- **游戏解锁** - 通过游戏获得的利润可以自由流通
- **单向消耗** - 空投池只出不进，总量持续减少

### 7.2 领取规则
- 每人可领：1,000 TPOT
- 总量：2亿 TPOT（20万人）
- 条件：先到先得，领完即止

### 7.3 使用流程
```
1. 领取空投 → 进入锁定账户（只能游戏，不能转账）
         ↓
2. 投入游戏 → 使用锁定的空投参与小时池/天池
         ↓
3. 游戏获利 → 中奖获得利润
         ↓
4. 利润释放 → 进入自由账户（可转账、交易、提现）
```

### 7.4 关键特点
- 空投本金始终锁定，只能通过游戏消耗
- 游戏利润可自由使用
- 防止空投直接砸盘
- 强制用户体验游戏
- 空投池单向消耗，不会回流
